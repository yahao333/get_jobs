# get_jobs

AI 驱动的求职自动化工具（开发中），目标是把“搜索岗位 → 过滤 → 生成打招呼语 → 投递简历”等流程串起来，减少重复操作成本。

## 功能概览（规划中）

- 多平台：Boss直聘优先，后续支持猎聘、前程无忧、智联招聘
- 搜索：按城市、关键词、薪资、经验、学历等条件筛选
- 过滤：公司/HR/职位黑名单、过滤不活跃 HR
- 打招呼：支持模板与 AI 个性化生成
- 投递：支持每日投递上限、可选发送图片简历
- 管理：计划提供 Web 管理界面

## 技术栈

- Go 1.25.0

## 配置

- 配置文件：config.yaml
- AI Key（按需设置）：
  - QWEN_API_KEY
  - MINIMAX_API_KEY
